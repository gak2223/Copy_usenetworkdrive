Copy_usenetworkdrive.bat

[概要]
・一時的にネットワークドライブを張ることによりネットワーク越しにコピーを作成します。
・コピーは指定のディレクトリの中にYYYYMMDD-HHMMSSの形でディレクトリをつくり、その下に作成されます。
・実行ユーザにネットワークドライブを張る権限やネットワークドライブを削除する権限が無い場合、ネットワークドライブの接続ユーザにディレクトリを作る権限がない場合には多分コピーは失敗します。

[動作環境]
・WindowsXP,Winows7での動作を確認しています。

[使用方法]
1.Copy_usenetworkdrive.batを任意のディレクトリにコピーしてください。
2.定義部分を修正して実行してください。
2.1.set COPYFROM="testdir"
　コピー元のディレクトリを指定してください。
2.2.set COPYTO="\\samba\share"
　コピー先に指定するネットワーク越しのディレクトリを指定してください。
2.3.set PASSWORD=PW
　コピー先に指定するネットワーク越しのマシンにログインするパスワードを指定してください。
2.4.set USERNAME=USERNAME
　コピー先に指定するネットワーク越しのマシンにログインするユーザ名を指定してください。
2.5.set DRIVENAME=Y
　一時的に作成するネットワークドライブのドライブレターを指定してください。
　必ず使っていないドライブレターを指定してください。
